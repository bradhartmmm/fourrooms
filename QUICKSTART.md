# 5-Minute Quickstart

Get from zero to magic. Right now.

---

## Step 1: Get Claude Pro or Max

Go to **https://claude.ai** and sign up for:
- **Claude Pro** ($20/mo) - Great for getting started
- **Claude Max** ($100/mo) - Unlimited usage

---

## Step 2: Download Claude Desktop

Go to **https://claude.ai/download**

Download and install it like any other app.

---

## Step 3: Download This Repo

1. Click the green **"Code"** button on GitHub
2. Click **"Download ZIP"**
3. Unzip it to your Desktop

---

## Step 4: Open in Claude Desktop

1. Open Claude Desktop
2. Click **"Choose Project"** (or drag the folder into Claude)
3. Select the **FR-Workflows** folder

---

## Step 5: Try the Magic

Type this in Claude:

```
/gamma "3 ways to automate your business this week"
```

**Watch the magic happen.** You'll get a live presentation URL in ~30 seconds.

---

## What Just Happened?

1. You typed a topic
2. Claude called the Gamma API
3. AI generated a full presentation
4. You got a shareable URL

**That's it.** No coding. No design. No slide deck building.

---

## Try More

**Grab a YouTube video:**
```
/absorb https://www.youtube.com/watch?v=VIDEO_ID
```

**Turn it into a workflow:**
```
/extract-workflow transcripts/video-title.txt
```

**Create a presentation from it:**
```
/gamma completed-workflows/video-title.prompt.md
```

**Get TikTok content ideas:**
```
/tiktok brainstorm
```

**Build your own workflow:**
```
/workflow start
```
